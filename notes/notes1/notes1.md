## What is Markdown?

 Markdown is a markup language created in 2004 by John Gruber that is written in basic text editor; while, have to use up less resource format options. Markdown uses it's unique file type called .markdown or .md for short.

 If you already use github in the past for whatever reason you have already stumbled on markdown before called, "~~README.md~~".
 
## What is Git?

Git is designed to keep track of a developer's progress and changes. Git is designed to handle any projects regardless of size.

## What is Github?

Github is coding hub that holds code from coders where it is held online.

They can also use it as back up if any unfortunate events occur like data corruption.

## What is Slack?

Slack is a social communication platform use similar to discord but for more work style environment; while, having tools that are basically shortcuts for apps useful for productivity.
